文件 → 十六进制转换器

将本地文件内容转换为十六进制字符串（纯客户端）

客户端处理（JavaScript，数据不会上传服务器）。将本地文件转换为十六进制字符串。 注意：文件大于 1MB 时可能导致资源消耗过高（如 Chromium 46 加载数 MB 的文本到 textarea 时有明显问题），大文件建议使用离线工具。

选项：

使用 0x 前缀及逗号分隔符（C 风格）
每 16 字节插入换行

选择文件： 或 拖放文件至此

输出：



另见：
bin2hex.exe（Windows） / bin2hex（Linux） （源码 bin2hex.c）
支持纯十六进制输出和自定义换行长度的版本：bin2hex.exe（bin2hex.c）

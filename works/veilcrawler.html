VeilCrawler · 网页采集器

Chrome 扩展 Manifest V3 · TypeScript · React · v1.0.1
解决「所见即不可得」的痛点
现代网页反爬机制复杂（参数签名、JS 混淆、环境检测），VeilCrawler 直接利用浏览器渲染好的 DOM 和已签名的网络请求，天然绕过反爬，无需逆向工程。

项目定位

一款轻量级可视化浏览器数据采集插件，基于 Chrome Manifest V3，以侧边栏沉浸式交互提供 DOM 选择 + JSON API 拦截双通道数据提取。 目标用户是无需编程基础的普通用户——点击网页元素即可自动生成提取规则，天然绕过反爬机制，无需逆向工程。

核心功能

可视化元素选择 鼠标点击页面元素自动生成 CSS/XPath 选择器，支持文本/HTML/链接/图片等多属性提取，智能高亮预览
JSON API 拦截 拦截页面 Ajax 请求，可视化浏览 JSON 响应结构，点击字段自动生成提取规则（适合前后端分离网站）
智能翻页 滚动翻页（无限滚动页面）+ 点击翻页（可视化选择"下一页"按钮），自定义间隔与最大采集量
数据清洗去重 默认智能去重，支持指定多字段联合主键，精准控制去重逻辑
多任务管理 创建/编辑/删除采集任务，配置自动保存到本地存储，支持导入/导出备份便于分享迁移
多格式导出 JSON / CSV 格式导出，支持一键复制到剪贴板

安装方式

下载 Release 安装包 (推荐)

前往 Releases 页面 下载最新版。
打开 chrome://extensions/ → 开启开发者模式 → 拖入 .crx 或加载解压的 .zip。

源码编译安装 (开发者)

git clone → pnpm install → pnpm build
加载 dist/ 目录到 Chrome 扩展页面。

Chrome 网上应用店

正在审核中，上线后可直接一键安装。

项目结构

目录职责

src/injected/XHR/Fetch 拦截脚本

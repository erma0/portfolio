标准速递 · 标准扫描工具

桌面应用 Python · FastAPI · pywebview · Playwright · v3.9.1

基于全国标准信息公共服务平台 (std.samr.gov.cn) 的安全标准批量扫描与下载工具， 覆盖国家标准 (GB)、行业标准 (HB)、地方标准 (DB) 三类标准。面向安全生产管理场景， 内置 113 个安全关键词自动匹配，支持验证码自动识别与预览画面拼图转 PDF。 提供桌面应用、WebUI 界面与 CLI 三种运行模式。

核心功能

三类标准全覆盖 国家标准 / 行业标准 / 地方标准，支持联合扫描
智能去重 os.scandir 高速扫描 + 多线程并行 + watchfiles 实时监控
验证码自动识别 ddddocr 驱动，自动重试 5-8 次，准确率极高
预览转 PDF Playwright 截图拼图块自动合成单页 PDF，无需人工操作
桌面应用 + WebUI pywebview 窗口 + 系统托盘，支持最小化到托盘；内置 FastAPI Web 界面
增量模式 已下载标准自动跳过，仅扫描新增内容，支持断点续传
多渠道通知 Server酱 / PushPlus / 企业微信 / 钉钉，扫描完成自动推送
定时任务 Cron 表达式定时扫描，支持启用/禁用/手动触发
任务控制 暂停/继续/重试，进度实时反馈，SSE 下载状态推送
113 安全关键词 消防/化工/机械/电气/特种设备/矿山/建筑/职业卫生/应急/交通/防护等
命令行全覆盖 CLI + GUI + API 三种模式，搜索/扫描/下载/增量全流程

标准下载说明

标准类型可下载说明
国家标准✅公开 PDF 下载，支持预览转 PDF
行业标准✅验证码下载，支持 AQ/XF/GA 等行业
地方标准✅验证码下载，支持各省地方标准
国家标准计划❌仅计划信息，无 PDF

标准下载流程（GB）

① 扫描 API 获取标准列表 (gbQueryPage) → ② 详情页提取 hcno (32 位 hex，与 API id 不同) → ③ 判断下载路径：直接下载 / 验证码下载 / 预览拼图 → ④ PDF 验证（content-type + %PDF- 魔数）→ 存入本地

⚠ 关键：API 返回的 id ≠ hcno，必须从详情页提取。检测到版权保护/ISO 标准则跳过。

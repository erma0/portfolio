发票酱 · 发票打印工具

桌面应用 Web 应用 Tauri 2 · Rust · pdf-lib · 纯前端

跨平台发票批量打印工具，支持 OFD / XML 数电票自动解析、PDF 矢量嵌入打印与批量模板管理。 提供两个独立版本，核心功能一致，覆盖不同使用场景：

桌面版 v2.0.8

master 分支 — 跨平台桌面安装包，性能优先

  • Tauri 2 底座，原生性能，安装包小巧（~8MB）
  • Rust 后端：OFD 解析、PDF 排版引擎、静默批量打印
  • 系统级打印能力：直接调用打印机驱动，支持静默输出
  • Windows / macOS / Linux 三平台安装包

技术栈：Tauri 2 · Rust · pdf-lib · print.js · Noto Sans SC

Web 版 v3.0.0

web 分支 — 纯静态前端，浏览器直接运行，零安装

  • 纯 HTML / CSS / JavaScript，无构建步骤，无服务端依赖
  • 浏览器内完成 OFD / XML 解析、PDF 生成、打印预览
  • 与服务端无交互，所有数据仅存于本地浏览器
  • 可部署到任意静态服务器，打开即用

技术栈：pdf-lib · pdf.js · ofd-client.js · xml-client.js · Local Font Access API

核心功能（两版通用）

OFD 解析

自动识别 OFD 格式数电票，提取发票号码、日期、金额、购销方等全部字段

XML 解析

支持 XML 格式数电票数据提取，兼容多版本 XML 结构

PDF 矢量打印

矢量 PDF 生成引擎，中文嵌入不失真。页面布局、水印、页脚、页码全可配

批量模板管理

多套打印模板切换，批量文件拖入一键处理，支持预览与单票微调

智能识别

自动检测发票类型、方向，匹配对应模板，减少手动操作

多格式导出

支持 PDF 直接打印、文件重命名、归档输出，适配财务归档流程

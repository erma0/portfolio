发票打印系统

桌面应用 Tauri 2 · Rust · pdf-lib · print.js

面向内部财务流程的跨平台桌面应用，覆盖 OFD / XML 数电票解析、PDF 矢量嵌入打印、批量模板管理等场景。 三个分支并行维护：master（v2.0.7 稳定版）、dev（开发版）、feat/web-cross-platform（v2.2.0 纯前端版）。

核心功能

  • OFD / XML 数电票自动解析与结构化提取
  • PDF 矢量嵌入打印：页面布局、水印、页脚、页码全支持
  • 批量模板管理与一键打印
  • 跨平台：Windows / macOS / Linux（Tauri 2 底座）
  • 纯前端版可脱离桌面环境，浏览器直接运行

技术栈

  • Tauri 2.x — Rust 桌面壳，轻量跨平台
  • pdf-lib — PDF 生成引擎，支持矢量中文字体嵌入
  • print.js — 打印逻辑核心，静默批量打印
  • Noto Sans SC — CJK 字体，保证打印效果一致

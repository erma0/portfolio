发票酱 · 发票打印工具

桌面应用 Web 应用 Tauri 2 · Rust · pdf-lib · PDF.js · 纯前端

跨平台发票批量打印工具，支持 PDF / OFD / XML 数电票 / 图片（JPG/PNG/BMP/WebP/TIFF）等多格式导入，智能排版，一键打印或导出。 提供两个独立版本，核心功能一致，覆盖不同使用场景：

桌面版 v2.0.8

master 分支 — Tauri 2 桌面应用，Windows 10 1803+ / 11

  • Tauri 2 + Rust 底座，原生性能，安装包 ~8MB
  • OFD 原生矢量渲染 + 发票信息直提，invoice-engine 独立 crate
  • PDF 渲染双引擎：WinRT 原生优先，自动 fallback PDFium（Chromium 内核）
  • PDF 文字层提取（轻量版无需 OCR）、PP-OCRv5 智能识别（OCR 版）
  • 图片格式支持：JPG / PNG / BMP / WebP / TIFF
  • 四种打印模式：PDF 阅读器 / 弹窗确认 / PDFium 静默 / SumatraPDF 静默
  • PDF 统一直通：lopdf Form XObject + JPEG DCTDecode，零质量损失
  • 深色/浅色模式切换
  • 快捷键：Ctrl+O 添加 · Ctrl+P 打印 · Ctrl++/- 缩放

技术栈：Tauri 2 · Rust · pdf-lib · printpdf · lopdf · invoice-engine · ocr-rs · WinRT · PDFium

轻量版 文字型 PDF/OFD 自动提取，PDF 文字层解析 ~5ms/页
OCR 版 PP-OCRv5 + MNN，文本优先+坐标回退，金额数学验证

Web 版 v3.0.0

web 分支 — 纯前端单页应用，零构建零后端，浏览器即开即用

  • 纯 HTML/CSS/JS，零依赖框架，无构建步骤
  • pdf-lib 1.17 embedPage 直接嵌入原始 PDF 页面，保留矢量

核心功能（两版通用）

OFD 完整支持

OFD 国标电子发票原生解析：矢量渲染、发票信息直提、印章保真、ImageMask 遮罩兼容，拖入即用

XML 数电票

解析 <EInvoice> 格式，提取发票号码/日期/金额/买卖方信息，汇总表与 CSV 全兼容

PDF 矢量打印

PDF 页面以原始质量嵌入合成，无二次压缩。桌面版 lopdf Form XObject 直通，Web 版 pdf-lib embedPage

自定义排版

纸张（A4/A5/B5/自定义）、布局 1-10×1-10 网格、边距间距缩放旋转、裁切线、水印、页码页脚

单票独立调整

每张发票预览拖拽移动+角落缩放、九宫格快速对齐、滚轮微调、±150mm 偏移、双击重置

批量模板管理

多套打印模板切换，批量文件拖入一键处理，文件列表记忆，打印状态追踪

发票汇总表

14 字段可勾选，金额/名称双击编辑，含税/不含税/税额自动汇总，CSV 导出 UTF-8 BOM

批量重命名

预设模板（金额+销售方+号码等）或自定义字段勾选，一键批量重命名发票磁盘文件

金额校验可视化

OCR/提取金额求和校验失败时金额徽章 ⚠ 警告，hover 查看含税/不含税/税额验证详情

多格式导出

PDF 直接打印 / 保存 / 批量重命名 / CSV 汇总导出，适配财务归档流程

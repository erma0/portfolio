抖音采集工具

已归档 Python · FastAPI · React · Vite · Aria2 · Playwright
⚠ 本项目已于 2026 年 6 月归档。
出于对法律法规的尊重，仓库原有代码已移除。以下内容为项目历史记录。

DouyinCrawler 是一个抖音公开数据采集工具，提供 GUI（pywebview）/ Web（React + Vite）/ CLI 三种运行模式。 以 Python 爬虫与 Aria2 WebUI 学习实践为初衷，功能覆盖抖音账号主页、喜欢、收藏、音乐原声、话题、搜索、合集、作品、关注、粉丝等公开数据的采集与批量下载。

采集范围

账号主页作品列表、置顶、基本信息
喜欢列表用户点赞的所有作品
收藏列表用户收藏的所有作品
音乐原声BGM 关联作品采集
话题/挑战赛话题下全部作品
搜索关键词搜索结果页内容
作品合集合集内作品批量
关注/粉丝社交关系数据
GUI 桌面pywebview 可视化界面
Web 控制台React + Vite 前端面板
CLI 模式命令行全功能
Aria2 下载集成批量下载引擎

技术栈

模块技术
运行模式GUI (pywebview) / Web (React + Vite) / CLI
后端框架Python + FastAPI
前端React + Vite + Tailwind CSS
下载引擎Aria2 WebUI 集成
浏览器驱动Playwright
部署Python 独立运行

备注

本项目仓库初衷为 Python 爬虫与 Web 技术的学习实践，源代码仅用于个人研究，从未用于商业用途。 项目原获 560+ Star，感谢曾经关注过的每一位。
